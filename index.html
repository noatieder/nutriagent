כדי להפעיל את המערכת, יש להזין מפתח API תקין של Groq.
המפתח נשמר מקומית בדפדפן בלבד ואינו מועבר לשרת כלשהו מלבד שרתי Groq.

🔒 המפתח מאוחסן ב-sessionStorage בלבד ונמחק עם סגירת הטאב.
לקבלת מפתח חינם: console.groq.com/keys
